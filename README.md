# UltimateZombieDefenseMOD
Ultimate Zombie Defense 16 players room MOD
# Content Description
1. Enhances gaming experience, supporting 16 players online;
2. Game difficulty automatically adjusts according to the number of players in the room;
3. After round 50, difficulty rises in waves while reasonably controlling the strength of zombies and bosses;
4. Increases weapon power, doubles bullet speed, and some weapons can automatically reload magazines;
5. Doubles player health, money, experience points (XP), building limits, and building health.
# How to use?
Download the zip file and extract it to the game's root directory.
